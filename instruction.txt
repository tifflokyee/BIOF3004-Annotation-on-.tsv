• For another VCF file, use this step-by-step workflow.

  1. Put the new VCF in this project folder
  Example:

  C:\Users\th0ma\OneDrive\Desktop\CAPNGSETB24 for BIOF\new_sample.vcf.gz

  2. Run the annotation pipeline up to gnomAD
  From PowerShell:

  cd "C:\Users\th0ma\OneDrive\Desktop\CAPNGSETB24 for BIOF"
  .\automation\run_stepwise_annotation.bat new_sample.vcf.gz

  This will create:

  result\step1_panelapp.tsv
  result\step2_revel.tsv
  result\step3_alphamissense.tsv
  result\step4_clinvar.tsv
  result\step5_gnomad.tsv

  It will stop before SpliceAI, because SpliceAI needs VEP web output.

  3. Create the small VCF for Ensembl VEP

  .\automation\create_small_vcf_for_vep.bat result\step5_gnomad.tsv -o small_clinical_variants_for_vep.vcf

  4. Upload this file to Ensembl GRCh37 VEP

  Upload:

  small_clinical_variants_for_vep.vcf

  Use this site:

  https://grch37.ensembl.org/Tools/VEP

  Enable:

  SpliceAI

  Then run VEP and download the result.

  5. Save the VEP result in this project folder
  For example:

  new_sample_vep_output.vcf

  6. Add the real SpliceAI results

  python .\automation\annotate_spliceai_step.py result\step5_gnomad.tsv -o result\step6_final_with_spliceai.tsv --vep-
  spliceai new_sample_vep_output.vcf

  Your final annotated file will be:

  result\step6_final_with_spliceai.tsv

  Important: do not reuse 5GFWIQAK7qxBoE7D.vcf_ for a different VCF/sample. That VEP file only matches the current
  dataset. For every new VCF, make a new small VCF, upload it to VEP, and use the new VEP result for SpliceAI.
